# Plataforma EAD – Êxito Ensino Jurídico

## Sobre a Empresa

A **Êxito Ensino Jurídico** é uma instituição dedicada à formação e preparação de estudantes e profissionais do Direito. Especializada em cursos preparatórios para exames como a OAB e concursos públicos, a Êxito se destaca pelo compromisso com a excelência no ensino e pela inovação no aprendizado jurídico.

## Sobre o Projeto

Este projeto tem como objetivo o desenvolvimento de um **Ambiente Virtual de Aprendizagem (LMS)** para otimizar a experiência dos alunos, oferecendo um ambiente moderno, interativo e intuitivo. A plataforma contará com recursos como:

✅ **Dashboard personalizado** com progresso do aluno  
✅ **Integração com sistema de pagamentos**  
✅ **Aulas ao vivo e gravadas** com materiais complementares  
✅ **Simulados interativos** e banco de questões detalhado  
✅ **Cronogramas dinâmicos** para organização dos estudos  
✅ **Área administrativa** para gestão completa dos cursos

O desenvolvimento será realizado utilizando **Angular** para o front-end e **Node.js** para o back-end, com hospedagem na **Google Cloud Platform (Firebase)**, garantindo escalabilidade, segurança e alto desempenho.

Esta documentação reúne todas as diretrizes, funcionalidades e requisitos do projeto, servindo como um guia para o desenvolvimento e a implementação da plataforma.
